AWS Tools
=========

My personal scripts I daily use to make some stuffs on AWS

Index
-----

### [epifes](./epifes)

Returns the Private IP of the EC2 instance for a given ECS service name and its cluster.

Usage:

```sh
./epifes myService production
#=> 10.50.12.99
```
